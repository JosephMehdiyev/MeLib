# Build
**WARNING!** I only tested this program for linux (arch).
To build and run, run these commands.
```
git clone https://github.com/JosephMehdiyev/MeLib
cd MeLib # the directory may vary, just cd to the project root
mkdir build
cd build
cmake ..
make
./melib
```
